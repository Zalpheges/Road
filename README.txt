ZQSD / souris : deplacement

leftclick : instantiation route
			Maintenir appuyé pour instantier plusieurs morceaux d'affilée
			
			Si appui sur un morceau existant, passage en mode Edition :
				R : Rotation 90° Clockwise du morceau de route
				Esc : Quitter le mode Edition
				Del : Supprimer le morceau de route 
				
Présence d'un bug dans le mode Edit : Les routes instanciées sur des emplacements où des routes ont précédemment été supprimées ne sont pas modifiables via le mode Edit.
